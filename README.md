# JSONPlaceholder Feed

Aplicación React que consume la API pública de JSONPlaceholder para mostrar publicaciones simuladas.

## Tecnologías
- React + Vite
- React Router DOM
- Shadcn/ui + Tailwind CSS
- JSONPlaceholder API

## Pasos para ejecutar

```bash
npm install
npm run dev
```

Luego abre http://localhost:5173
